\doxysubsection{Detailed Description}
Absztrakt ősosztály a karakterenként működő titkosítási algoritmusokhoz. 

Az osztály egységes interfészt biztosít különböző titkosítási algoritmusok számára. A konkrét algoritmusok ebből az osztályból származnak.
